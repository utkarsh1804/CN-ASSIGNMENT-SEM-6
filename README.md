# Computer Network Lab (Cisco Packet Tracer)

## Overview

This repository contains my Computer Network lab work done using Cisco Packet Tracer. I have implemented different networking concepts to understand how networks actually work in real scenarios.

## Experiments

* Network Topologies (Star, Ring, Mesh, Tree)
* LAN setup using Hub and Switch
* Ping and Tracert commands
* Static IP configuration
* IP addressing and subnetting
* Wireless network setup
* End-to-end network design

## Usage

You can download the .pkt files and open them in Cisco Packet Tracer to check the configurations and simulations.
